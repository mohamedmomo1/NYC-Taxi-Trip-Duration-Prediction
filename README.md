# NYC Taxi Trip Duration Prediction 🚕
This project predicts the total ride duration of taxi trips in New York City using **Ridge Regression**.

## Key Achievements:
- **MAE:** 290.64 seconds (~4.8 minutes).
- **RMSE:** 433.99 seconds.
- Cleaned and processed over 900k rows of data.
- Handled outliers and performed feature engineering (Haversine distance, hour/day extraction).

## Technologies Used:
- Python (Pandas, NumPy, Scikit-learn)
- Visualization (Matplotlib, Seaborn)
